# Movio PR #10 — Complete Customer Journey

## Changelog

- End-to-end customer booking path: request → offers → payment → confirmation → trip status → ratings
- Live route estimate on new trip (distance + duration) with Google Maps embed
- Premium offer cards with driver/vehicle photos, ratings, experience, languages, Elite badge, ETA, response time
- Offer sorting: lowest price, highest driver rating, highest vehicle rating, fastest response
- Accept offer no longer auto-confirms — routes to Stripe-prepared payment page
- Platform fee stays server-side (hidden from customer checkout UI)
- Premium confirmation page with booking reference, driver, vehicle, map, ETA
- Journey tracker: searching → offers → selected → paid → en route → arrived → in progress → completed
- Driver ETA field on offer form; open requests show distance/duration
- Post-trip reviews for both driver and vehicle
- Demo seed includes distance/duration, ETA minutes, and vehicle ratings

## New features

- `/pedidos/novo` route preview before publish
- `/pedidos/[id]/pagamento` checkout (demo Stripe path)
- `/pedidos/[id]/confirmacao` booking confirmation
- `OfferCards` premium marketplace UI
- `JourneyTracker` status stepper
- `GET /api/routes/estimate`
- Stripe provider scaffolding (`PAYMENTS_ENABLED`)
