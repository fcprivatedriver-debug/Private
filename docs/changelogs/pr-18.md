# PR — Revisão V1 experiência inicial

## Changelog

Contas novas começam **totalmente vazias** (sem receitas, despesas, categorias, gráficos ou objetivos). Dados de exemplo ficam só no modo Demo interno. Receitas manuais, despesas editáveis, seletor de contas redesenhado, Guia no menu, lista de compras com várias listas, e área RGPD.

## Novas funcionalidades / correções

- Registo sem categorias nem movimentos automáticos
- Categorias criadas só quando o utilizador as usa
- Separador Receitas com tipos (Salário, Subsídio, Donativo, …)
- Editar / eliminar receitas e despesas
- Dashboard com empty states e CTAs de arranque
- Seletor Pessoal / Familiar redesenhado (mobile-friendly)
- Menu inferior: Guia
- Lista de compras com várias listas (criar, editar, eliminar, partilhar)
- Privacidade / RGPD: exportar, apagar histórico, faturas, conta
- FAB e espaçamentos ajustados para evitar sobreposições
