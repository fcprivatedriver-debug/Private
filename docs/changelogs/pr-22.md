## Update — ecrã branco Vercel

**Causa:** `@prisma/adapter-neon` ignorava `?schema=nina` e usava o schema `public` (Zrik). Exceções server-side → ecrã branco / Application error.

**Correção:** `new PrismaNeon({ connectionString }, { schema: "nina" })` + seed demo se BD vazia no build.
