# PR — Nina PWA Versão 1

## Changelog

Preparação da Nina para utilização diária como Progressive Web App: instalação no ecrã principal, ecrã inteiro, atalhos rápidos e captura por voz imediata.

## Novas funcionalidades

- Manifest oficial **Nina** (standalone)
- Ícones + splash
- Service Worker (cache inteligente + atualização automática)
- Página offline
- Atalhos: Dashboard, Falar com a Nina, Fotografar Fatura, Lista de Compras, Objetivos
- Captura rápida `/pt/captura?mode=voice&auto=1`
- Guia de instalação em Mais (Android + iPhone)
- Navegação mobile: Nina · Compras · Falar · Objetivos · Mais
