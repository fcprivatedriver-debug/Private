# PR — Filosofia e missão final da Nina

## Changelog

A missão oficial passa a orientar toda a experiência: *“A vida é para ser vivida. A Nina trata das contas.”* Landing, shell, captura, chat, insights e sugestões alinhados a esta visão — assistente diária que retira preocupação, antecipa e simplifica.

## Novas funcionalidades

- Módulo canónico `src/lib/ai/mission.ts` + `docs/MISSION.md`
- Landing com missão como herói e canais voz/texto/foto
- Sidebar, meta, PWA e dashboard com a missão
- Sugestões proativas: automatizar tarefas repetitivas + reforçar poupanças quando há folga
- Insights IA com lembrete de automatização
- Captura e perfil com copy alinhada à missão
