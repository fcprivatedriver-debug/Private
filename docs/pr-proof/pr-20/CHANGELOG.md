# PR #20 — Edição, autor, auditoria e permissões

## Changelog

- Edição completa de receitas e despesas (valor, descrição, categoria, data, conta Pessoal/Familiar) e eliminação
- Ficha do movimento ao tocar na lista: histórico + editar + guardar + eliminar
- Autor visível em cada movimento na Conta Familiar
- Histórico de alterações (criado por/em, última alteração por/em + timeline)
- Perfil: «Como pretendes que a Nina te trate?»
- Definições da Conta Familiar: permitir (ou não) editar movimentos uns dos outros
- Seed demo alinhado com as categorias de receita atuais

## New features

- Corrigir um movimento sem duplicar (ex.: Pingo Doce 11 € → 21 €)
- Transparência familiar: quem criou e quem alterou cada registo
- Nome preferido usado em toda a app e no histórico
- Permissão familiar configurável para edição cruzada
- Atualização automática de saldos, dashboard e estatísticas após cada alteração
